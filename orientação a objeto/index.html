FAQ (Perguntas Frequentes):

    Oque é orientação a objetos (OO)
    A orientação a objetos (OO) é uma forma de programação que se baseia na ide Propriedades (atributos) : São como características ou detalhes sobre o objeto. Imagine que você tem um objeto chamado "carro". Algumas propriedades desse carro podem ser de sua cor, marca e modelo. Métodos (funções) : São as ações que o objeto O JavaScript, como muitas outras linguagens de programação, suporta a Sem JavaScript, um OO Um exemplo simples em JavaScript seria: javascript function Carro(marca, modelo, cor) { this.marca = marca; this.modelo = modelo; this.cor = cor; this.ligar = function() { console.log("Vrumm!"); }; } const meuCarro = new Carro("Toyota", "Corolla", "Azul"); console.log(meuCarro.cor); // Retorna "Azul" meuCarro.ligar(); // Exibe "Vrumm!"
